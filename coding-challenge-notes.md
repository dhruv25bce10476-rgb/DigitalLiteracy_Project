## Platform Chosen: HackerRank (hackerrank.com)

### About HackerRank
HackerRank is a competitive programming and technical skill assessment platform widely used by companies for job recruitment and by students for practicing problem solving. It offers challenges across domains which include algorithms, data structures, Python, Java, SQL, and more.

### Challenge Completed
- **Challenge Name:** Solve Me First
- **Domain:** Algorithms → Warmup
- **Difficulty:** Easy
- **Language Used:** Python

### Challenge Description
The "Solve Me First" challenge was an introductory problem where you are given two integers and must return their sum. While simple, it teaches the fundamentals of reading input, processing data, and producing output in the HackerRank environment.

### Solution Approach
```python
def sum(a,b):
    c=a+b
    return c
a=int(input())
b=int(input())
print(sum(a,b))
```

### What I Learned
This challenge helped me understand how online judges work,how input is read from standard input, processed through a function, and the output is validated against expected results. It gave me confidence to attempt more complex problems next.
